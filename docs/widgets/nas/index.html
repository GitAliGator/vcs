      Complete NAS Interactive Exploration Guide

      🎯 Quick Test Scenarios
      • • "Perfect Alignment": See baseline scores with optimal chronological order
      • • "Minor Disorders": Small swaps affecting both NAS-D and NAS-L
      • • "Rotate Half": Splits narrative in middle, tests structural integrity
      • • "Neighbor Swaps": Adjacent sentence swaps affecting flow smoothness
      • • "Major Misalignment": Dramatic reordering showing worst-case scenarios
      • • "Reverse Order": Complete narrative reversal
      🎛️ Interactive Controls
      • • Drag Purple Markers: Adjust alignment positions to see real-time NAS-D and NAS-L changes
      • • LCT Slider: Control tolerance for minor deviations in both distance and flow calculations
      • • Hover Markers: See individual penalty values for NAS-D distance calculations
      • • Watch Pipeline: Observe how each component contributes to the final score
      📊 Advanced Chunk Manipulation

      Simulate Deleted Content:

      Simulate Added Content:
      • • Click to deactivate chunks from Reference text
      • • Represents extra narrative elements in generation
      • • Observe how precision and recall metrics respond
      Reorder Narratives: Drag segments within lists to create custom chronological scenarios and see comprehensive scoring changes across all three components.
      🔬 What to Observe
      • • NAS-D sensitivity: How distance penalties increase with chronological displacement

      • • Window Regularizer: How coverage efficiency affects the final regularized score
      • • Component interactions: How different misalignment types create distinct score profiles
      • • LCT effects: How tolerance adjustments impact both distance and flow calculations

      What's Next: Video Comprehension Score (VCS)

      While Complete NAS provides comprehensive narrative alignment assessment, it doesn't capture semantic content quality. VCS addresses this by evaluating the actual content coverage and semantic fidelity:

      • • Content Coverage: Measures how well generated text covers the semantic space of reference content
      • • Semantic Fidelity: Evaluates the quality and accuracy of semantic information transfer
      • • Content Density: Assesses information density and avoiding semantic dilution
      • • Complementary Assessment: Works alongside NAS to provide complete narrative and content evaluation
