Window Regularizer: Preventing Over-Permissive Assessment

Purpose & Design

The Window Regularizer prevents scenarios where mapping windows become excessively large, making narrative assessment too permissive. It measures the ratio of total window coverage to timeline area and penalizes over-coverage.

Key Benefit: Maintains assessment stringency by penalizing overly generous mapping windows.

Calculation Method

Compares total mapping window area against timeline area (TR × TG), normalized by minimum possible area (1/max(TR,TG)). Values range from 0 (no penalty) to 1 (maximum penalty).

Formula: (Coverage Ratio - Min Area) / (0.5 - Min Area)
